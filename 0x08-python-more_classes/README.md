Task 0: Simple rectangle
Define an empty class Rectangle.

Task 1: Real definition of a rectangle
Define a class Rectangle with private instance attributes width and height, and property getters and setters for them. Also define an __init__() method with optional width and height parameters.

Task 2: Area and Perimeter
Define public instance methods area() and perimeter() that return the area and perimeter of the rectangle, respectively.

Task 3: String representation
Define special methods __str__() and __repr__() that return string representations of the rectangle, which can be used to print it or recreate a new instance using eval().

Task 4: Eval is magic
Make the __repr__() method return a string representation of the rectangle that can be used to recreate a new instance using eval().

Task 5: Detect instance deletion
Define a special method __del__() that prints the message "Bye rectangle..." when an instance of Rectangle is deleted.
